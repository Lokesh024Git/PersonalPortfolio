Travel Plan Creator using Generative AI

This project uses Streamlit to create an interactive web application for generating travel itineraries.

It integrates via the ‘google.generativeai’ library to generate detailed itineraries based on user-provided inputs such as the destination city, trip duration, and daily schedule preferences.

The utilization of generative AI, namely Google's PaLM2 API, seeks to transform the way we organize.

This project uses a Random Forest Classifier to predict hotel booking cancellations based on guest details and booking characteristics.

Then receive instant cancellation predictions, demonstrating practical application of machine learning in hotel management.
